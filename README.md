### `Features`


* Implemented with the newest Apollo version (v4).
* Graphql Shield is applied.
* Subscriptions is applied.
* File Uploading.
* Advanced Pagination.
* Advanced Sorting.
* Advanced Searching.
* Advanced Filtering.
* Above Advanced Features can be combined together and its results reflected on pagination.
* Authentication.
* Authorization.
